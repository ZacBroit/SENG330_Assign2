Detailed Description

Sub Class Treadmill.

Treadmill is a sub class of GymMachine

Treadmill's have one unique variable which is their maximum speed for the belt Treadmill's also work with the Prototype design pattern, so a clone function is used to create a new instance of the class based off of the pre-existing prototype referred to as 'tread'
